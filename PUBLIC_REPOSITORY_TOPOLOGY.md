## Public Topology Principles
- Public tree is conceptual-first and implementation-agnostic.
- Public demo is cognition surface, not runtime execution surface.
- NotebookLM ingests only curated public doctrine (`docs/notebooklm_public/`).

## Relocation Rule
Everything outside this topology must be reviewed and either:
- relocated to private repository scope, or
- rewritten into conceptual/public-safe form before retention.

## Forbidden Topology in Public
- runtime execution directories
- orchestration/control-plane directories
- enforcement substrate code paths
- replay/verification implementation internals
