# DETERMA Public Repository Scope

## Purpose

This repository exists as a public governed execution showcase.

It demonstrates:

- governed execution concepts
- constrained mutation flows
- replay blocking behavior
- append-only lineage concepts
- execution authority boundaries
- public runtime demonstrations

---

# Public vs Private Architecture

The public repository intentionally contains only a limited proof slice.

The complete DETERMA authority architecture is maintained privately.

Private materials may include:

- deep protocol semantics
- authority issuance internals
- replay invalidation internals
- mathematical and cryptographic specifications
- governance internals
- internal orchestration architecture
- partner diligence material
- roadmap and sequencing documents
- internal NotebookLM corpus

---

# Responsible Disclosure Model

The public repository is intended for:

- technical evaluation
- architectural discussion
- ecosystem exploration
- runtime demonstration
- governed execution education

It is not intended to expose the complete internal authority system.

---

# Access Requests

Strategic, enterprise, research, and technical stakeholders may request curated private access for deeper architectural review.

Access is evaluated individually.
