# Purpose

This document defines the canonical operating model for developing DETERMA using the Factory.

The purpose is NOT fully autonomous development.

The purpose is:

```text
high-speed bounded autonomous execution
```

The Factory exists to compress implementation cycles while preserving:

- governance discipline
- architectural consistency
- operational boundaries
- doctrine alignment
- fail-closed execution behavior

---

# Core Principle

The Factory is NOT an autonomous architect.

The Factory is:

```text
bounded implementation acceleration
```

The Factory must NOT:

- redefine architecture
- redefine product strategy
- expand platform scope
- create generalized abstractions
- recursively redesign systems

The Factory SHOULD:

- implement bounded tasks
- generate tests
- repair CI
- perform plumbing work
- execute repetitive implementation
- maintain documentation consistency
- generate governed PRs

# Canonical Operational Stack

## Layer 1 — Doctrine

Human-owned.

Canonical doctrine documents remain the source of truth.

The Factory may NOT override doctrine.

---

## Layer 2 — Planning

Human-led.

Humans define:

- execution priorities
- task ordering
- architectural constraints
- operational boundaries

---

## Layer 3 — Task Decomposition

Semi-autonomous.

The Factory may assist with:

- task breakdown
- implementation grouping
- execution sequencing

But only inside canonical doctrine constraints.

---

## Layer 4 — Implementation

Highly autonomous.

The Factory should autonomously:

- implement code
- generate tests
- repair integration issues
- refactor bounded systems
- update replay structures
- synchronize docs

---

## Layer 5 — Verification

Mixed human + Factory responsibility.

The Factory performs:

- automated tests
- replay verification
- contract validation
- mutation verification

Humans perform:

- doctrine validation
- architecture validation
- legitimacy review
- release decisions

---

## Layer 6 — Authority Release

Human-controlled.

Execution legitimacy authority must remain external to autonomous cognition.
