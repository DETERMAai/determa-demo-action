# DETERMA Canonical Tagging Rollout Plan

STATUS: CANONICAL
EXPOSURE: INTERNAL_CORE

## Purpose

This document defines the operational rollout plan for applying canonical governance metadata across existing DETERMA materials.

The objective is to:

- reduce ambiguity
- support repository governance
- standardize disclosure handling
- improve corpus curation
- preserve lineage visibility

---

# Canonical Principle

```text
Governance metadata should exist on real documents, not only governance policies.
```

---
