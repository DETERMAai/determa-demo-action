# DETERMA Repository Segmentation Model

## Purpose

This document defines the canonical segmentation model for DETERMA repositories.

The objective is to:

- preserve disclosure boundaries
- reduce accidental exposure
- separate proof from mechanism
- support scalable governance
- maintain continuity

---

# Canonical Principle

```text
Different audiences should interact with different knowledge layers.
```

---

# Repository Segments

## Segment 1 — Public Showcase

Purpose:

- awareness
- onboarding
- positioning
- category framing
- design partner discovery

Contains:

- public-safe demos
- public positioning
- sanitized architecture
- onboarding flows
- public FAQ

Must not contain:

- restricted internals
- patents
- cryptographic semantics
- deep replay mechanics

---

## Segment 2 — Curated Diligence

Purpose:

- investor onboarding
- partner evaluation
- technical orientation
- enterprise review

Contains:

- sanitized runtime docs
- selected architecture
- selected threat scenarios
- runtime philosophy
- curated governance materials

Must remain:

- disclosure-aware
- sanitized
- curated

---

## Segment 3 — Internal Core

Purpose:

- implementation continuity
- canonical runtime semantics
- governance continuity
- operational doctrine

Contains:

- runtime internals
- governance semantics
- orchestration semantics
- database doctrine
- canonical contracts

---

## Segment 4 — Restricted Archive

Purpose:

- invention preservation
- moat protection
- strategic continuity

Contains:

- patent-sensitive material
- cryptographic internals
- governance mutation semantics
- replay invalidation internals
- strategic sequencing

---

# Segmentation Rules

## Rule 1

Public material should demonstrate outcomes rather than implementation depth.

---

## Rule 2

Curated diligence should maximize understanding while minimizing moat exposure.

---

## Rule 3

Restricted archive material should only be exposed intentionally.

---

## Rule 4

NotebookLM corpora should follow repository segmentation boundaries.

---

# Canonical Principle

```text
Segmentation preserves clarity, continuity, and strategic leverage.
```
