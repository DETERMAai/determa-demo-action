# NotebookLM Source Set

This directory is the curated public source set for NotebookLM.

Allowed NotebookLM focus:
- why runtime drift matters
- why approvals become stale
- why governed execution exists
- why replay can become invalid
- why mutation legitimacy must be re-evaluated

Disallowed disclosure scope:
- exact replay invalidation implementation
- capability token mechanics
- cryptographic enforcement specifics
- operational bypass resistance details

Only documents in this folder should be ingested into NotebookLM.
