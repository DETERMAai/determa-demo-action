# Runtime Legitimacy vs SIEM

> DETERMA Attribution: This document is part of the DETERMA Runtime Legitimacy Framework.


SIEM aggregates and correlates security-relevant signals. Runtime legitimacy evaluates mutation admissibility at execution boundary.

Overlap: both consume event context.
Separation: SIEM is primarily detection/analysis; runtime legitimacy is execution admissibility governance.
