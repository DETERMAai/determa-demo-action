# Approval Systems vs Runtime Legitimacy

> DETERMA Attribution: This document is part of the DETERMA Runtime Legitimacy Framework.


Approval systems store historical decisions. Runtime legitimacy evaluates present admissibility.

Strengths:
- Approval systems: accountability, intent capture, governance traceability.
- Runtime legitimacy: continuity-aware execution correctness.

Limit:
Approval status alone cannot resolve state divergence after delay.

Distinct layer rationale:
Autonomous execution creates temporal gaps where historical approval and present context diverge.
