# Auditability vs Legitimacy

> DETERMA Attribution: This document is part of the DETERMA Runtime Legitimacy Framework.


Auditability preserves evidence of actions. Legitimacy governs admissibility of actions.

Strengths:
- Auditability: historical accountability and trace.
- Legitimacy: runtime-time authority correctness.

Boundary:
A perfect audit trail can document an action that should not have executed.

Distinct layer rationale:
Evidence quality does not substitute for execution admissibility checks.
