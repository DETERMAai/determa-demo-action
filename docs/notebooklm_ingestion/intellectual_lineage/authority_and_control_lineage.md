# Authority and Control Lineage

Control theory and governance disciplines distinguish command from admissible action under system constraints. Legitimity theory extends this distinction into autonomous mutation systems.

Historical precedent:
- controller authority bounded by plant state
- supervisory control with safety overrides
- command admissibility contingent on current conditions

Extension:
Approval expresses intent; legitimacy determines executable authority under current runtime state.

Why inevitable:
Systems with high autonomy require authority models that are state-aware, temporally bounded, and externally constrained.
