# Zero Trust Lineage

Zero Trust reframed perimeter assumptions into continuous verification. Legitimacy theory parallels this move for mutation authority.

Historical precedent:
- no implicit trust from network location
- continuous verification of access context
- assumption minimization under dynamic conditions

Extension:
Legitimity doctrine applies continuous verification to execution admissibility. Historical approval is not a standing trust perimeter.

Why inevitable:
Runtime state changes as rapidly as identity context; authority must be continuously re-anchored.
