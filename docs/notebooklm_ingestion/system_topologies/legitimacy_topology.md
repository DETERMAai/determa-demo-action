# Legitimacy Topology

Legitimacy topology maps how approval, time, drift, and authority interact.

```text
APPROVAL
↓
EXECUTION DELAY
↓
RUNTIME DIVERGENCE
↓
LEGITIMACY UNCERTAINTY
↓
REVALIDATION
↓
INVALIDATION
```

The topology emphasizes that legitimacy is a path property, not a point property.
