# Current Runtime Capabilities

The current DETERMA public framework demonstrates several operational runtime legitimacy capabilities.

---

## Demonstrated Capabilities

### Runtime State Comparison

The framework compares execution conditions across different runtime moments.

### Runtime Divergence Detection

The demos demonstrate how execution legitimacy may collapse after runtime mutation.

### Stale Authorization Identification

The framework identifies situations where historical authorization no longer matches current runtime conditions.

### Replay Legitimacy Failure Modeling

DETERMA demonstrates why replayable approval artifacts are not automatically legitimate under changed runtime states.

### Mutation Visibility

The demos expose how runtime mutation changes execution legitimacy assumptions.

### Execution Evidence Generation

The framework generates structured execution evidence explaining:

- what was approved
- what changed
- what diverged
- why legitimacy failed

### Legitimacy Revalidation Logic

The framework models legitimacy re-evaluation after runtime change.

### Contradiction Walkthroughs

The repository includes guided contradiction demonstrations showing legitimacy collapse flows.

---

## Current Scope

The current framework is focused on:

- legitimacy modeling
- contradiction exposure
- ontology stabilization
- runtime governance reasoning
- execution legitimacy semantics

It is not yet positioned as a complete production runtime governance platform.
