# EXCLUDED FROM PUBLIC INGESTION

The following categories are intentionally excluded from `docs/notebooklm_public` ingestion:

- runtime substrate internals
- orchestration mechanics
- replay implementation details
- verification internals
- operational sequencing
- factory workflow
- execution topology implementation
- deployment mechanics

## Why Excluded

Public ingestion is optimized for conceptual legitimacy doctrine.

Excluded categories increase implementation leakage risk and collapse the public/private boundary between field cognition and operational substrate.
