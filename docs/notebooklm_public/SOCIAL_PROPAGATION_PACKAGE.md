# Social Propagation Package

## Purpose

Provide a consistent public sharing flow for the runtime legitimacy category.

- `assets/runtime_legitimacy_contradiction.gif`
- `assets/visuals/what_determa_has_proven.png`

## Recommended Share Order

1. Contradiction GIF
2. One-sentence contradiction line
3. Live demo link
4. What has been proven visual
5. Board-level one pager link

## X / LinkedIn Compression Guidance

- Lead with contradiction, not product features.
- Keep first post to one insight:
  - "The approval remained. The system changed."
- Include one asset and one link max per post.
- Use a second post for proof visual and live demo.

## Investor Intro Guidance

- First link: `BOARD_LEVEL_ONE_PAGER.html`
- Second link: live demo
- Third link: NotebookLM for deep dive

## GIF Usage Guidance

- Use as top-of-thread context setter.
- Keep accompanying copy under 2 short paragraphs.

## Screenshot Usage Guidance

- Use `what_determa_has_proven.png` for credibility framing.
- Use `runtime_sequence_card.svg` for conceptual clarity.

## Demo Sharing Flow

1. Share the one-pager.
2. Ask viewer to run the live demo.
3. Ask viewer to open execution evidence in demo.
4. Route to `WHY_DETERMA_EXISTS.html`.
5. Offer NotebookLM for extended reasoning.
