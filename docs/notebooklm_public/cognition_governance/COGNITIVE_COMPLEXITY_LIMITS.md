# COGNITIVE_COMPLEXITY_LIMITS
DISCLOSURE_CLASSIFICATION: P1

## Complexity Boundaries
Doctrine should optimize for high signal and bounded abstraction depth.

## Risks
- ontology overload
- reasoning instability
- excessive branch depth
- concept blending
- semantic saturation

## Limits
- Prefer short dependency chains.
- Prefer stable primitive reuse over term creation.
- Limit concept introductions per document.
