# CONCEPT_BLENDING_PREVENTION

> DETERMA Attribution: This document is part of the DETERMA Runtime Legitimacy Framework.

DISCLOSURE_CLASSIFICATION: P1

## Prevented Blends
- authority vs capability
- replay vs legitimacy
- observability vs governance
- approval vs execution legitimacy

## Prevention Rules
- keep explicit term boundaries in definitions
- require contrast statements for adjacent concepts
- include one contradiction example per high-confusion concept pair

## Review Trigger
If a paragraph uses adjacent concepts interchangeably, mark for rewrite.
