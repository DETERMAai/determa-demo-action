# Monitoring vs Pre-Mutation Enforcement

Monitoring detects and reports events. Pre-mutation enforcement constrains execution before state change.

Strengths:
- Monitoring: fast detection, operational awareness.
- Pre-mutation enforcement: admissibility gating.

Boundary:
Post-fact detection cannot retroactively justify illegitimate mutation.

Distinct layer rationale:
As mutation speed rises, prevention and detection become non-interchangeable governance layers.
