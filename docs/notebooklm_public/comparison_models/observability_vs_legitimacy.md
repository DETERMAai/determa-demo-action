# Observability vs Legitimacy

> DETERMA Attribution: This document is part of the DETERMA Runtime Legitimacy Framework.


Observability explains what happened. Legitimacy determines what may happen.

Strengths:
- Observability: diagnosis, correlation, post-event clarity.
- Legitimacy: pre-mutation admissibility under current runtime continuity.

Boundary:
Observability can confirm execution; it cannot by itself prove execution should have occurred.

Why legitimacy emerges:
Autonomous mutation requires decision quality at execution boundary, not only post-fact visibility.
