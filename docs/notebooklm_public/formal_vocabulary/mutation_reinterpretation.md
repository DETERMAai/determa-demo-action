# Mutation Reinterpretation

> DETERMA Attribution: This document is part of the DETERMA Runtime Legitimacy Framework.


Mutation reinterpretation occurs when unchanged action instructions map to different operational effects due to runtime evolution.

Implications:
- Action identity does not fix action meaning.
- Approval of prior interpretation does not automatically authorize new interpretation.

Contradiction:
Same mutation string, different state interaction semantics.

Distributed-systems connection:
Comparable to schema evolution changing semantics of unchanged queries.
