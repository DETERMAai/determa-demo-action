# Temporal Correctness Lineage

> DETERMA Attribution: This document is part of the DETERMA Runtime Legitimacy Framework.


Temporal correctness appears across systems disciplines: leases expire, deadlines matter, stale reads invalidate writes. Legitimacy doctrine is a continuation of this tradition for execution authority.

Historical precedent:
- lease-based leadership validity
- expiration of assumptions over time
- temporal preconditions in safety-critical control

Extension:
Approval is treated as time-indexed evidence. Execution legitimacy is evaluated at the present boundary, not frozen at approval time.

Why inevitable:
Autonomous execution magnifies delay and drift, making temporal correctness of authority unavoidable.
