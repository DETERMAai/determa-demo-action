# ONTOLOGY_EXPANSION_CONTROLS
DISCLOSURE_CLASSIFICATION: P1

## Admission Controls
New concepts are admitted only when:
- existing primitives cannot express the idea
- dependency mapping is explicit
- conceptual compression is preserved
- boundary impact is documented

## Mandatory Tests
- duplication detection
- primitive dependency validation
- contradiction differentiation
- vocabulary stabilization check

## Compression Constraints
- Prefer extension of canonical terms over new naming.
- Prefer one precise concept over multiple adjacent variants.
- Reject concepts that increase ontology load without structural gain.
