# Market Positioning

DETERMA is positioned as Runtime Legitimacy Infrastructure for Autonomous Systems.

## Category Definition

DETERMA addresses a distinct control gap:

Historical approval persists, but Runtime Authority conditions can change before execution.

DATP is the foundational runtime authority model that closes this gap operationally.

## What DETERMA Is Not

- not a generic observability stack
- not a static approval workflow
- not an agent orchestration replacement
- not a policy-only layer detached from execution-time state

## Practical Outcome

DETERMA enables:

- fail-closed execution under Authority Continuity failure
- bounded runtime mutation authority
- immutable authority lineage for verification
- clearer operational governance for autonomous execution
