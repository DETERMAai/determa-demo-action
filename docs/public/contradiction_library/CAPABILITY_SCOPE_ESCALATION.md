# Capability Scope Escalation

## Operational Setup

Approval was issued for limited runtime capability scope.
Execution is attempted under elevated capability scope.

## Contradiction

Historical approval still appears valid.
Authority bounds changed before execution.

## Authority Continuity Outcome

Capability scope transition breaks Authority Continuity for the original grant.

## DATP Framing

DATP enforces bounded execution authority.
When scope exceeds approved continuity bounds, execution is denied.
