# Replay Under New Context

## Operational Setup

A prior execution artifact is replayed in a newer runtime context.

## Contradiction

Replay identity is stable.
Runtime legitimacy context is not.

## Authority Continuity Outcome

Replayability does not preserve Authority Continuity.

## DATP Framing

DATP separates replay identity from Runtime Authority admissibility.
Replay proceeds only when continuity is re-established.
