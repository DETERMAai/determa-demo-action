# Sandbox Reallocation Mismatch

## Operational Setup

Approval occurred in sandbox context S1.
Execution resumes after sandbox reallocation to S2.

## Contradiction

Authorization token persists.
Execution substrate assumptions changed.

## Authority Continuity Outcome

Execution context transition breaks continuity between approved and active runtime state.

## DATP Framing

DATP applies fail-closed execution when Runtime Authority cannot be carried across reallocation boundaries.
