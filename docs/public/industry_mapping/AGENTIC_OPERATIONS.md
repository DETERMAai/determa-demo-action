# Agentic Operations Mapping

## Typical Flow

An autonomous agent receives approval for a bounded action.
Execution resumes later with expanded or shifted runtime capability context.

## Contradiction Pattern

- approved intent remains
- runtime authority surface changed
- mutation can exceed original continuity assumptions

## DATP Operational Interpretation

- Bounded execution authority is revalidated at mutation time.
- Authority Continuity failure triggers fail-closed execution.
