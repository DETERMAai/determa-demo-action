# Runtime Demo Surface

This folder is the public runtime demo surface.

Entrypoints:
- UI: `ui/runtime_dashboard.html`
- API shell: `runtime/api_shell.py` (public demo wrapper)

Purpose:
- demonstrate governed execution concepts and runtime legitimacy mismatch behavior
- support public-facing walkthroughs and design-partner demonstrations

Note:
Operational enforcement internals are intentionally excluded from the public knowledge surface.
